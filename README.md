## webpack demo
### 目录结构
<pre>
.
├── README.md
├── bundle                   // 项目build目录
├── bower_components         // 通过 bower 安装的第三方库
├── package.json             // 项目配置文件
├── webpack.config.js        // webpack 配置文件
├── app                      // 生产目录
│   ├── components           // 各种组件
│   ├── views                // 各种页面
│   ├── router.js            // 路由配置
│   ├── App.vue              // 根组件
│   ├── main.js              // Webpack 预编译入口
│   └── index.html           // 项目入口文件
</pre>
